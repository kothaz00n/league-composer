## 2025-02-12 - [Composition Analysis Hot Loop O(N^2) Optimization]
**Learning:** In `src/engine/recommend.js`, the `getCompositionAnalysis` loop for finding substitution candidates checked `Object.values(teamRoles).includes(candidateName)` inside a hot loop evaluating all champions (`allNames`). In JS, `Object.values` creates a new array and `.includes` performs a linear search, creating an O(N^2) complexity path for analyzing substitutions across a 160+ champion pool.
**Action:** When performing repeated membership checks across large data pools inside hot loops, always pre-calculate a `Set` outside the loop. Refactored to `const teamMemberNames = new Set(Object.values(teamRoles));` and replaced `.includes()` with `.has()`, nearly doubling execution speed for the analysis batch.
